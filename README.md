Daily Quote App
A simple and user friendly Quote Android App. The app dispaly random motivational and inspirational quotes 
that allow user to refresh, share with others and save favorite quotes.

1. Setup Instruction (Android Studio)
   Prerequisites:
   Android Studion (Android Meerkat)
   Android SDK
   Internet Connection (fro API)

   a. Clone Repository
   b. Open in Android Studion (Open in Android Studio -> click Open Project -> Select the project -> Let Gradle Sync complete)
   c. Run the app ( Connect an Android Device -> click Run -> App will launch on your android device)

2. AI Coding Approach
   I created this project using an AI-first development workflow to improve speed, code, quality and productivity.

   My Approach:
   I follow many approach which are following as -
   a. First of used AI to understand app architecture (Clean Architecture + MVVM + Repository)
   b. Generate UI design using Stitch (An AI tool by Google for UI design)
   c. Used AI to write api and local database (roomdb) Integration.
   d. Debugged error and crashes comes while running and implementing functionality.

3. AI Tools Used
   While building the proeject i used following AI tools to inhance code structure and protect from error/bugs.
   a. ChatGPT (Code GPT 5.2) - Architecture design & code generation
   b. Gemini - Debugging 
   c. Stitch - AI-generated UI design tool.
